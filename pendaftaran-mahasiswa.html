Google Apps Script — Web App
Tempel kode ini di Google Apps Script untuk menerima data dari form dan menyimpannya ke Sheets.
// ── Google Apps Script (Code.gs) ──
// Tools > Script Editor > paste ini > Deploy as Web App

function doPost(e) {
  const sheet = SpreadsheetApp
    .openById("15GexwSBcTJrUj-Nx6D0PIsmq4-hZ4LRMI8UWuAxPS50")
    .getActiveSheet();

  const data = JSON.parse(e.postData.contents);
  const row = [
    data.noReg, data.nama,     data.nik,
    data.email, data.wa,       data.prodi1,
    data.jalur, data.nilai,    data.sekolah,
    new Date().toLocaleString("id-ID")
  ];

  sheet.appendRow(row);

  return ContentService
    .createTextOutput(JSON.stringify({ status: "ok", noReg: data.noReg }))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet() {
  return ContentService
    .createTextOutput("PMB API aktif ✓");
}

Cara Setup & Deploy ke GitHub Pages
Ikuti langkah berikut untuk mempublikasikan website pendaftaran secara online gratis.
1. Buat Repository github.com → New Repository → "pmb-2025" → Public
2. Upload File Upload index.html dan assets ke repo
3. GitHub Pages Settings → Pages → Branch: main → Save
4. URL Live https://[username].github.io/pmb-2025
5. Custom Domain Settings → Pages → Custom domain → pmb.universitasmu.ac.id
